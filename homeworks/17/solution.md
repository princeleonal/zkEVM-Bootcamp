| row 	| inc 	| acc 	|
|:---:	|:---:	|:---:	|
|  0  	|  0  	|  0  	|
|  1  	|  5  	|  5  	|
|  2  	|  6  	|  11 	|
|  3  	|  2  	|  13 	|
|  4  	|  1  	|  14 	|
|  5  	|  0  	|  14 	|
|  6  	|  9  	|  23 	|

|  8  	|  3  	|  32 	|
| ... 	| ... 	| ... 	|
| 116 	|  8  	| 308 	|
| 117 	|  6  	| 314 	|
| 118 	|  7  	| 321 	|
| 119 	|  1  	| 322 	|

$A_{1,1}$ is 1

$A_{6,2}$ is 9 (increment)

$A_{6,3}$ is 23 (accumulator)

## Constraints
- $A_{0,2}$ = 0 and $A_{0,3}$ = 0
- All $1 >= i <= 119$ 
    - $0 >= A_{i,2} < 10$
    - $A_{i,3} - A_{i,2} - A_{i-1,3} = 0$
